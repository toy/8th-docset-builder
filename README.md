# 8th docset

Build requirements:
* `curl` to get html
* `ruby` (with `nokogiri`) to prepare
* `dashing` to build docset
* `tar` to archive

Build instructions:
```shell
git clone https://github.com/toy/8th-docset-builder.git
cd 8th-docset-builder
make
```
